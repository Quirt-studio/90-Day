原型说明

产品概述

Career Compass Pro 是一款AI驱动的职业规划平台，专为20-30岁年轻专业人士设计。 通过AI垂直领域定位、动态技能热力图、90天原子级计划等核心功能， 帮助用户精准定位职业方向，缩短迷茫期，降低试错成本。

技术栈

HTML5 Tailwind CSS FontAwesome Responsive Design Web Platform

原型导航

首页 (Home)

产品主页，展示核心价值和功能入口

登录注册 (Authentication)

用户认证界面，支持多种登录方式

AI垂直领域定位器 (Locator)

AI驱动的职业评估和领域推荐

动态技能热力图 (Heat Map)

实时技能需求分析和个人技能对比

90天原子化计划生成器 (Planner)

个性化学习路径规划和进度跟踪

实时职位解构引擎 (Analyzer)

职位需求分析和技能匹配度评估

风险收益计算器 (Calculator)

职业转换风险评估和收益预测

个人中心 (Profile)

用户档案、学习进度和个人设置
